# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
  - リモートリポジトリ: インターネット上で共有するリポジトリ
  - ローカルリポジトリ: ローカル環境上に存在するリポジトリ

## プッシュとマージの違いは何でしょうか？
  - プッシュ: ローカルリポジトリ上のコミットをリモートリポジトリにUPすること
  - マージ: あるブランチのコミットをmainブランチなどに結合すること

## コミットとプッシュの違い
  - コミット: ローカルリポジトリに変更履歴を保存すること、そのデータ
  - プッシュ: コミットをリモートリポジトリに共有すること

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
  - 行った変更がわかりやすいように、簡潔に書く


## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
  - ローカルでマージする場合、コミットした後、ローカルリポジトリでマージしてプッシュする
  - プルリクエストでマージする場合、コミットした後プッシュした後、リモートリポジトリでプルリクエストを作成し、承認されるとマージされる

## コンフリクトを起こしてしまった場合、どう対処すべきですか？
  - 双方の変更点を反映させるかどうかを双方で確認する
  - 前者の変更のみを反映、後者の変更のみの反映、双方の変更を反映を決定する。
